The goal of this repository is to prove out that (a) probablistic programming beats out (b) linear regression for the particular (1) daily, (2) monthly correlation 
that arises from monthly EIA and daily weather correlation. 

This is meant to be a public demonstration that Spectral Technologies' statistical model beats out linear regression.
